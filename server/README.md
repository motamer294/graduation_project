# Help Desk Ticketing System
Professional graduation project built with **Django 6.0**, **PostgreSQL**, and **Docker**.

## Quick Start
1. Create virtual environment: `python -m venv .venv`
2. Activate it: `source .venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`
4. Run Docker: `docker-compose up -d`
5. Migrate DB: `python manage.py migrate`
6. Run Server: `python manage.py runserver`

## Team Rules
- Always `git pull` before starting.
- Keep `.venv` out of the repository.
